# তাসবিহ কাউন্টার (Tasbih Counter)

এটি একটি প্রেয়ার-ফোকাসড তাসবিহ কাউন্টার ওয়েব অ্যাপ, যেখানে নির্দিষ্ট নামাজের সময় অনুযায়ী যিকির কাউন্ট করা যায়। অ্যাপটি অফলাইনে কাজ করে এবং PWA হিসেবে মোবাইল/ডেস্কটপে অ্যাপের মতো ইনস্টল করা যায়।

## এই প্রজেক্টের মূল উদ্দেশ্য

- নামাজভিত্তিক যিকির ট্র্যাক করা
- নির্দিষ্ট সময়ের বাইরে ভুল কাউন্ট কমানো
- দৈনিক প্রেয়ার প্রগ্রেস ও স্ট্রিক ধরে রাখা
- অফলাইনেও ব্যবহারযোগ্য একটি হালকা অ্যাপ দেওয়া

## প্রধান ফিচারসমূহ

- প্রেয়ার-ভিত্তিক মোড (Prayer Mode)
- কাস্টম মোড (Custom Mode)
- একাধিক দুয়া/যিকির অপশন ও টার্গেট
- স্ট্রিক সিস্টেম
- ভাইব্রেশন ফিডব্যাক ও ইনটেনসিটি কন্ট্রোল
- কাউন্টার স্টাইল পরিবর্তন (Classic/Modern)
- ফুল স্ক্রিন মোড
- সম্পূর্ণ ডাটা রিসেট অপশন
- PWA সাপোর্ট (ইনস্টল + অফলাইন ক্যাশ)

## কীভাবে ব্যবহার করবেন

1. অ্যাপ খুললে বর্তমান নামাজ সময় অনুযায়ী প্রেয়ার সিলেক্ট হবে।
2. দুয়া ড্রপডাউন থেকে পছন্দের যিকির নির্বাচন করুন।
3. মাঝের বড় বাটন/এরিয়ার উপর ট্যাপ করলে কাউন্ট বাড়বে।
4. টার্গেট পূর্ণ হলে দুয়ার পাশে সম্পন্ন চিহ্ন দেখা যাবে।
5. Settings থেকে প্রয়োজনমতো মোড, স্টাইল, ভাইব্রেশন, ফুলস্ক্রিন পরিবর্তন করুন।

## Prayer Mode এ কাউন্ট কখন কাজ করে, কখন করে না

Prayer Mode এ কাউন্ট খুব নিয়ন্ত্রিতভাবে কাজ করে:

- শুধু চলমান নামাজের সময়েই কাউন্ট হবে
- অন্য প্রেয়ার সিলেক্ট থাকলে কাউন্ট হবে না
- প্রেয়ার সময় শেষ হলে ওই প্রেয়ারের কাউন্ট লক হয়ে যায়
- কোনো প্রেয়ার উইন্ডোর বাইরে থাকলে কাউন্ট হবে না

ডিফল্ট প্রেয়ার টাইম উইন্ডো:

- ফজর: 05:00 - 07:00
- যোহর: 12:00 - 15:00
- আসর: 16:30 - 17:30
- মাগরিব: 18:00 - 19:00
- ইশা: 20:00 - 22:00

নোট: Prayer Mode এ আপনি যদি ভাবেন কাউন্ট হচ্ছে না, বেশিরভাগ ক্ষেত্রে কারণ হয় বর্তমানে সক্রিয় প্রেয়ার-উইন্ডো মেলেনি।

## Custom Mode কীভাবে কাজ করে

Custom Mode এ প্রেয়ার টাইম লক প্রযোজ্য নয়।

- যেকোনো সময় কাউন্ট করা যায়
- Custom target (10-1000) সেট করা যায়
- ডাবল ট্যাপ করলে কাস্টম কাউন্ট 0 হয়ে যায়

এটি ফ্রি কাউন্টিং বা ব্যক্তিগত টার্গেটের জন্য উপযোগী।

## Full Screen মোড

Settings > Screen > Full Screen চাপলে ফুল স্ক্রিন চালু হবে।

- আবার চাপলে Exit Full Screen হবে
- কিছু ব্রাউজারে ইউজার-জেসচার ছাড়া Full Screen ব্লক হতে পারে

## Reset Data (সব ডাটা রিসেট)

Settings থেকে Reset all data চাপলে:

- সব প্রেয়ার কাউন্ট মুছে যাবে
- স্ট্রিক মুছে যাবে
- সেভ করা সেটিংস ডিফল্টে ফিরে যাবে
- কাস্টম মোড ডাটাও মুছে যাবে

এটি স্থায়ীভাবে ডাটা মুছে ফেলে।

## শুধু দৈনিক Prayer Data কীভাবে রিসেট হয়

অ্যাপে আলাদা Daily Prayer Reset বাটন নেই। Daily reset স্বয়ংক্রিয়ভাবে হয়:

- নতুন দিন শুরু হলে (date change) প্রেয়ারভিত্তিক কাউন্ট ও সম্পন্ন দুয়া তালিকা রিসেট হয়
- স্ট্রিক আলাদা লজিকে ধরে রাখা হয়
- Custom mode target/count আগের মতো থাকতে পারে

অর্থাৎ, প্রতিদিনের প্রেয়ার ডাটা ম্যানুয়ালি নয়, স্বয়ংক্রিয়ভাবে দিনের পরিবর্তনে রিসেট হয়।

## স্ট্রিক কীভাবে কাজ করে

স্ট্রিক বাড়ার জন্য শর্তগুলো:

- ইশা সময় বা তার পরে স্ট্রিক চেক হয়
- ইশাতে অন্তত 1টি দুয়া সম্পন্ন থাকতে হবে
- ৫টি প্রেয়ার প্রতিটিতে ন্যূনতম 3টি করে দুয়া সম্পন্ন থাকতে হবে
- একই দিনে একবারই স্ট্রিক আপডেট হয়

স্ট্রিক আপডেট লজিক:

- আগের স্ট্রিক দিনের পরের দিন হলে: streak +1
- ধারাবাহিক না হলে: streak = 1

## দুয়া সম্পন্ন স্ট্যাটাস

- প্রতিটি দুয়ার টার্গেট পূরণ হলে সেটি সম্পন্ন ধরা হয়
- সম্পন্ন দুয়াতে চেকমার্ক দেখা যায়
- অসম্পন্ন থাকলে ক্রস চিহ্ন দেখা যায়

## PWA ফিচার ও অ্যাপ হিসেবে ইনস্টল

এই প্রজেক্টে web manifest + service worker আছে, তাই এটি ইনস্টলযোগ্য অ্যাপ হিসেবে ব্যবহার করা যায় এবং প্রয়োজনীয় ফাইল অফলাইনে ক্যাশ হয়।

### Android (Chrome)

1. ওয়েবসাইট খুলুন
2. ব্রাউজার মেনুতে যান
3. Add to Home screen বা Install app চাপুন
4. হোমস্ক্রিন থেকে অ্যাপের মতো চালু করুন

### Desktop (Chrome/Edge)

1. সাইট খুলুন
2. অ্যাড্রেস বারের Install আইকনে ক্লিক করুন (বা মেনু থেকে Install)
3. Install নিশ্চিত করুন
4. অ্যাপ উইন্ডো হিসেবে চালু হবে

### iPhone/iPad (Safari)

1. Safari দিয়ে সাইট খুলুন
2. Share বাটন চাপুন
3. Add to Home Screen নির্বাচন করুন
4. হোমস্ক্রিন থেকে চালু করুন

নোট: iOS-এ PWA আচরণ Android/Desktop এর তুলনায় কিছু সীমিত হতে পারে।

## অফলাইন ব্যবহার

একবার অ্যাপ লোড হলে গুরুত্বপূর্ণ ফাইলগুলো ক্যাশে থাকে। পরে নেটওয়ার্ক না থাকলেও বেসিক ব্যবহার চালিয়ে যেতে পারবেন।

## কার জন্য উপযোগী

- যারা নামাজভিত্তিক যিকির শৃঙ্খলাভাবে করতে চান
- যারা স্ট্রিক দিয়ে ধারাবাহিকতা ট্র্যাক করতে চান
- যারা মোবাইলে আলাদা অ্যাপের মতো দ্রুত ব্যবহার চান

## সংক্ষিপ্ত সতর্কতা

- Prayer Mode ইচ্ছাকৃতভাবে সময়-নিয়ন্ত্রিত, তাই সবসময় কাউন্ট নাও হতে পারে
- Reset all data করার আগে নিশ্চিত হোন, কারণ এটি অপরিবর্তনযোগ্য
